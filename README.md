# Escape Room
Desenvolvimento de Escape Room para projeto final de Rede de Computadores I.

## Requisitos
- Python

## Como rodar? (Linux)
Depois de instalar os documentos deste repositório, abra ao menos *três* terminais em janelas diferentes. 
1 - _python3 server.py_.
2 - _python3 client.py_ para o primeiro usuário. 
3 - _python3 client.py_ para o segundo usuário.
